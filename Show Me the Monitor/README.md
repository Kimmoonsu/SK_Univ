Application Name : Show Me The Monitor

Show me the monitor는 온라인에서 회의, 스터디, 튜터링 등을 제공하는 프로그램입니다.

방장이 방을 만들게 되면 참가자들이 입장합니다. 참가자들의 화면이 방장에게 보여지고,

방장이 특정 화면을 클릭하게 되면 모두가 같은 화면을 보게 됩니다.

음성과 텍스트 채팅을 가능하게 합니다.

Client : C# Windows Form

Server : C#, TCP, UDP
