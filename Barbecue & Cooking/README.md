이름 : 김문수

소속 : 서경대학교

Java Swing을 이용한 게임입니다.

Application Name : Barbecue & Cooking

Barbecue & Cooking은 동서남북 4방향에 대각선 방향까지 합하여 8방향을 검사하는 게임입니다.

기존의 애니팡은 4방향을 검사하여 제거하지만, Barbecue & Cooking은 대각선까지 검사하여 아이템을 제거합니다.

마우스와 키보드를 이용하여 게임이 가능합니다. 

MultiThread를 이용하여 게임의 움직임을 강조하였습니다.

혼자하기와, 둘이하기 모드를 지원합니다.

개발 환경 : Windows 7 64bit, Eclipse LUNA, JDK7

개발 언어 : JAVA
